# Shred n Breakfast

Archive site for the 2009 ski web series (Next.js App Router).

## Deploy (Vercel)

1. Connect this repo in the [Vercel dashboard](https://vercel.com/new) (import from GitHub).
2. Add environment variables from `.env.example` under **Settings → Environment Variables** (Production / Preview as needed).
3. Deploy uses the default Next.js build: `npm run build`.

Local dev: `npm install` and `npm run dev`. Copy `.env.example` to `.env.local` for Bunny Stream API access.
